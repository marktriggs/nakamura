This is a binary distribution of Sakai Nakamura based on Sling.

Before you start Sakai Nakamura, you should look at system.cfg. At the
current time this is the only file that you should need to edit.

You should be able to start Sakai Nakamura now, by running ./startup.sh


Then point your browser to http://localhost:8080/

The default admin username is `admin` and password is `admin`
